Instant Messaging

What it is and its Protocols

What is Instant Messaging?

Instant Messaging is a form of real-time back-and-forth digital communication, usually between two people, via an application or software.

What are some Instant Messaging Protocols?

  • IRC Protocol:

      • - Internet Relay Chat is a protocol that allows users to participate in conversation in group discussions, specifically through channels.

  • WebRTC Protocol:

      • - Web Real-Time Communication is a framework that allows users to communicate through multimedia formats, such as video and audio.

  • WebSocket Protocol:

      • - WebSocket is a protocol that allows user computers to communicate and transmit directly with a server through a stable two-way HTTP connection. It allows stable and direct connection from the core to the edge.
